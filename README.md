### Automation project
- Java, Selenium WebDriver, Allure, JUnit, Cucumber, Cypress
- JUnit tests
- Cucumber tests
- Cypress tests

##### Run JUnit tests:
`./gradlew test` - local run
`./gradlew testSelenoid` - selenoid run

##### Open JUnit tests Allure report:
`./gradlew allureServe`

##### Run Cucumber tests:
`./gradlew cucumber`

##### Run cypress:
`npm run cypress`
